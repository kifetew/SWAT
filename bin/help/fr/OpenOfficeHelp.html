Utiliser les bibliographies JabRef dans OpenOffice.org

JabRef peut exporter votre base de données dans le format de tableur OpenOffice.org 1.1 .sxc ainsi que dans le format OpenDocument .ods utilisé par OpenOffice.org 2.

Dans les deux cas, la table exportée contiendra une feuille de calcul, listant les entrées selon les lignes et les différents champs en colonnes. L'ordre et les noms des colonnes sont configurés pour être compatible avec les fonctions bibliographiques d'OpenOffice.org (OOo 1.1 dans le cas d'une exportation au format .sxc, et OOo 2 pour une exportation au format .ods).

En fonction de la version d'OpenOffice.org, voici la façl;on d'utiliser les bibliographies JabRef :

Utiliser le fichier exporté comme une base de données bibliographique dans OpenOffice.org 2.3 et 2.4

Suivez les étapes suivantes pour configurer une feuille de calcul exportée par JabRef comme une base de données bibliographique dans OpenOffice.org :

  • Depuis JabRef, exporter votre base en utilisant le format .ods
  • Démarrer OpenOffice.org Writer
  • Choisir Edition -> Changer de bases de données. Cliquer sur Parcourir, puis sélectionner votre base exportée au format .ods.
  • Dérouler le + placé devant le nom de la base importée puis cliquer sur le nom du fichier qui apparaît, et enfin sur le bouton Définir.
  • Choisir Outils -> Options -> OpenOffice.org Base -> Base de données. Dans cette fenêtre, la base de données que vous avez importée doit apparaître, ainsi que la base de données bibliographique par défaut (Bibliography).
  • Éditer la base de données Bibliography, et lui donner un autre nom, tel que Bibliography-old (en effet OpenOffice.org Writer ne permet pas de choisir entre plusieurs bibliographies).
  • Sélectionner votre base de données bibliographique, l'éditer et la renommer Bibliography en faisant attention à la graphie du mot (majuscule en début de nom).
Après avoir effectué ces étapes, votre base de données bibliographique devrait être prête à être utilisée avec OpenOffice.org. Pour le vérifier, sélectionner Insertion -> Index -> Entrée de bibliographie... : la liste de clés BibTeX doit apparaître.

Utiliser le fichier exporté comme une base de données bibliographique dans OpenOffice.org 2.0, 2.1 et 2.2

Suivez les étapes suivantes pour configurer une feuille de calcul exportée par JabRef comme une base de données bibliographique dans OpenOffice.org :

  • Exporter votre base en utilisant le format .ods
  • Démarrer OpenOffice.org Writer
  • Choisir Outils -> Options -> OpenOffice.org Base -> Base de données
  • Éditer la base de données Bibliography, et lui donner un autre nom, tel que Bibliography-old (en effet OpenOffice.org Writer ne permet pas de choisir entre plusieurs bibliographies)
  • Fermer la fenêtre Options et choisir Fichier -> Nouveau -> Base de données
  • Choisir Se connecter à une base de données existante, sélectionner Classeur comme type de base de données, cliquer sur Suivant et choisir le fichier .ods que vous avez exporté.
  • Cliquer sur Terminer, puis choisir le nom Bibliographie quand demandé.
Après avoir effectué ces étapes, choisir Outils -> Base de données bibliographique. Votre base de données devrait maintenant s'afficher.

Utiliser le fichier exporté comme une base de données bibliographique dans OpenOffice.org 1.1.x

  • Exporter votre base de données vers le format .sxc
  • Démarrer OpenOffice.org Texte
  • Choisir Outils -> Sources de données...
  • Sélectionner la base de données Bibliography et lui donner un autre nom, tel que Bibliography-old. Cliquer sur Appliquer.
  • Cliquer sur Nouvelle source de données. Une nouvelle entrée apparaît. La renommer Bibliography.
  • Changer le type de BD en Classeur. Cliquer le bouton ... au bout de la ligne URL de la source de données. Choisir le fichier .sxc que vous avez exporté.
  • Cliquer sur OK pour fermer la fenêtre Gestion des sources de données.
Après avoir effectué ces étapes, choisir Outils -> Base de données bibliographique. Votre base de données devrait maintenant s'afficher.
